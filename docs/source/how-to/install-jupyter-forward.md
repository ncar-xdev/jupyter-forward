# Install jupyter-forward

jupyter-forward can be installed in three ways:

```{eval-rst}
.. tab:: pip

    Using the `pip <https://pypi.org/project/pip/>`__ package manager:

    .. code:: bash

        $ python -m pip install jupyter-forward

.. tab:: conda

    Using the `conda <https://conda.io/>`__ package manager that comes with the
    Anaconda/Miniconda distribution:

    .. code:: bash

        $ conda install jupyter-forward --channel conda-forge

.. tab:: Development version

    To install a development version from source:

    .. code:: bash

        $ git clone https://github.com/NCAR/jupyter-forward
        $ cd jupyter-forward
        $ python -m pip install -e .

```
